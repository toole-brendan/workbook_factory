# DDG sheets package

`ddg.sheets` is the consolidated registry for the DDG-51 workbook.

The first refactor keeps the proven DDG-sliced TAM/SAM sheet implementations as
compatibility modules and composes them through this single registry.  The split
executive-summary tabs are replaced by one merged `Executive Summary` sheet; the
remaining TAM and SAM analytical sheets are ordered by workbook group rather than
by source workbook half.
